# ADMCHECK - Checklist Admissional (Mobile)

![Version](https://img.shields.io/badge/version-1.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)
![Mobile](https://img.shields.io/badge/platform-mobile--first-orange)

Sistema mobile-first para controle de checklist de documentos admissionais, desenvolvido com HTML, CSS e JavaScript puro. Ideal para uso em smartphones no setor de RH/DP.

## ✨ Funcionalidades

- ✅ Cadastro de funcionários com matrícula e nome
- ✅ Checklist completo categorizado (obrigatórios, escolaridade, registros, dependentes)
- ✅ Status visual por cores (verde 100%, amarelo parcial, cinza vazio)
- ✅ Persistência local com localStorage
- ✅ Sidebar horizontal adaptada para mobile
- ✅ Exportar relatório .txt
- ✅ Copiar relatório para área de transferência
- ✅ Modal de confirmação estilizado
- ✅ Interface dark slim e responsiva

## 🚀 Tecnologias

- HTML5 semântico
- CSS3 puro (mobile-first, flexbox, grid)
- JavaScript Vanilla (ES6)
- localStorage para persistência
- Ícones com emojis nativos

## 📱 Como usar

### Acesse online
[ADMCHECK_MOBILE/](https://lucasfreire99.github.io/ADMCHECK_MOBILE/).

### Instalação local
1. Clone o repositório
2. Abra o arquivo `index.html` no navegador
3. Todos os dados ficam salvos no seu dispositivo

### Funcionamento básico
1. Preencha matrícula e nome
2. Clique em "Criar"
3. Marque os documentos conforme recebidos
4. Clique em "Salvar" para persistir
5. Use os botões laterais para exportar ou copiar
